Galeri Project Mahasiswa

ROBOT PENGHINDAR HALAMAN OTOMATIS

Robot mobil penghindar halangan ini berhasil berfungsi sesuai rancangan. Sensor ultrasonik mampu mendeteksi objek dan mengirim data ke Arduino, sehingga robot dapat mengambil keputusan secara otomatis untuk bergerak atau menghindar. Project ini membuktikan bahwa sistem berbasis mikrokontroler dapat digunakan untuk membuat kendaraan otonom sederhana yang responsif terhadap lingkungannya.

ROBOT PENGHINDAR HALAMAN OTOTMATIS

Project ini berhasil menunjukkan bagaimana sistem robotik dapat bergerak secara mandiri tanpa kendali manusia, dengan bantuan sensor ultrasonik untuk mendeteksi jarak objek di depannya. Melalui pengaturan logika pada mikrokontroler Arduino, robot dapat menghindari rintangan secara efisien. Ini membuktikan prinsip dasar autonomous navigation dapat diaplikasikan secara sederhana dan efektif menggunakan sensor jarak dan motor DC.

MINIATUR RUMAH PINTAR(SMART HOME)

Project ini membuktikan bahwa konsep otomatisasi rumah (smart home) dapat diimplementasikan dengan teknologi sederhana berbasis Arduino. Sistem mampu mengontrol perangkat seperti lampu dan kipas berdasarkan data dari sensor, serta meningkatkan keamanan menggunakan sensor gerak. Kesimpulannya, project ini berhasil menampilkan contoh nyata penerapan Internet of Things (IoT) dalam lingkungan rumah tangga modern.

SISTEM KEBAKARAN OTOMATIS

Project ini berhasil merancang alat pendeteksi dini kebakaran yang bekerja otomatis dengan sensor suhu atau api. Sistem mampu memberikan peringatan berupa bunyi alarm ketika suhu melewati batas aman. Dengan demikian, alat ini dapat menjadi solusi sederhana dan murah untuk meningkatkan keamanan lingkungan dari risiko kebakaran. Kesimpulannya, project ini berhasil menggabungkan sistem monitoring dan peringatan otomatis dengan akurasi cukup baik.

TRAINER ROBOTIK ARDUINO

Project ini berfungsi dengan baik sebagai media pembelajaran interaktif untuk memahami dasar robotika dan pemrograman mikrokontroler Arduino. Melalui kombinasi sensor, motor, dan tombol, pengguna dapat mempelajari cara kerja input-output secara langsung. Kesimpulannya, trainer ini efektif digunakan untuk praktikum dan pelatihan dasar sistem kontrol otomatis berbasis Arduino.
